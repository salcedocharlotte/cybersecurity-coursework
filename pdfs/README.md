# PDF Resources

This folder is for sharing PDF files related to cybersecurity coursework, notes, and study materials.

## How to Contribute

1. **Fork** this repository.
2. Add your PDF file(s) to this `pdfs/` directory.
3. Update the index below with the file name and a brief description.
4. Open a **Pull Request** with your additions.

## Guidelines

- Only upload PDFs that you own or have permission to share.
- Keep file names descriptive and use hyphens instead of spaces (e.g., `network-security-notes.pdf`).
- Do not upload files larger than 100 MB (GitHub file size limit).
- Ensure content is relevant to cybersecurity coursework or study.
